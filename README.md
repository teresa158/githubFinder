# 🔍 GitHub Finder

Une application web simple et responsive développée avec **HTML, CSS et JavaScript**, qui permet de rechercher des profils GitHub, afficher leurs informations et consulter leurs repositories.

---

## ✨ Fonctionnalités

- 🔎 Rechercher un utilisateur GitHub par son **username**
- 👤 Afficher les informations du profil :
  - Avatar
  - Nom
  - Bio
  - Followers / Following
  - Nombre de repositories
- 📦 Afficher la liste des repositories
- ⭐ Trier les repos par nombre d’étoiles (stars)
- 💾 Sauvegarde des favoris dans le **localStorage**
- 🗑️ Supprimer un favori facilement
- 📱 Design responsive (mobile + desktop)

---

## 🛠️ Technologies utilisées

| Technologie | Utilisation |
|-----------|-------------|
| HTML5 | Structure du site |
| CSS3 | Style + Responsive design |
| JavaScript (Vanilla) | Logique, DOM, API |
| GitHub REST API | Récupération des données |
| localStorage | Sauvegarde des favoris |
